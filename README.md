# react-ts-news-viewer

## 프로젝트 개요
React + TypeScript + Vite로 만든 간단한 뉴스 뷰어

## 기능 목록
- 카테고리(전체보기, 비즈니스, 건강 등)별 뉴스 기사 조회
- 카테고리 클릭 시 URL 변경되고 해당 뉴스 목록 자동 렌더링
- NavLink를 활용한 CSR 방식 라우팅

## 기술 스택
- HTML
- CSS
- TypeScript
- Vite
